# 🖊️ AI Blog Generator with Multi-Language Translation

An AI-powered blog writing tool built with **Google Gemini 2.5 Flash** and **Gradio**. 
Generates a full blog post (Intro, Fun Facts, Advantages, Conclusion) based on 
topic, audience, and tone — with optional instant translation into 9 languages.

## ✨ Features
- Custom topic, audience, tone, and word count
- AI-generated structured blog content
- One-click translation into Hindi, Telugu, Tamil, Spanish, French, German, Arabic, Chinese, Japanese
- Clean Gradio web interface

## 🛠️ Tech Stack
- Python
- Google Gemini API (`gemini-2.5-flash`)
- Gradio

## ⚙️ How to Run Locally
1. Clone this repo
2. Install dependencies: `pip install google-genai gradio`
3. Set your Gemini API key as an environment variable: `GEMINI_API_KEY`
4. Run the notebook / script

## 📌 Note
This started as a base blog generator and was extended with multi-language 
translation support as an added feature.
